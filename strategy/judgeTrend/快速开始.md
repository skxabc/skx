# 快速开始指南

## 当前问题

免费数据源（akshare、yfinance）目前遇到网络问题，无法获取数据。

## 解决方案：使用Tushare（推荐）

### 步骤1：注册Tushare账号（免费）

1. 访问：https://tushare.pro/register?reg=1
2. 注册账号（免费版即可使用）
3. 登录后，在"接口TOKEN"页面获取您的token

### 步骤2：使用Token运行系统

**方式1：通过命令行参数**
```bash
cd /Users/user/skx/skx/strategy/judgeTrend
python3 main.py --market all --tushare-token YOUR_TOKEN
```

**方式2：通过环境变量（推荐）**
```bash
# 设置环境变量（每次打开终端都需要设置，或添加到~/.zshrc）
export TUSHARE_TOKEN=YOUR_TOKEN

# 运行系统
cd /Users/user/skx/skx/strategy/judgeTrend
python3 main.py --market all
```

**方式3：永久设置环境变量**
```bash
# 添加到 ~/.zshrc 文件（macOS）
echo 'export TUSHARE_TOKEN=YOUR_TOKEN' >> ~/.zshrc
source ~/.zshrc

# 然后直接运行
python3 main.py --market all
```
